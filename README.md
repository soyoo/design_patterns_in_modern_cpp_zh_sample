# 译者序

作者：Dmitri Nesteruk

译者：郑书原

邮箱：soyoo@aliyun.com

正如 Linux 的创始人 Linus Torvalds 在 2000年8月25日给 linux kernel 邮件列表的一封邮件中提到的：Talk is cheap, show me the code。本书通过以代码讲解的方式，将设计模式在现代C++中的使用方法呈现出来，给人耳目一新的感觉，有效的进一步理解设计模式本身。

献给最爱的 C++ ~
